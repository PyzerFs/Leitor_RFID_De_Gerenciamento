# Leitor_RFID_De_Gerenciamento
Sistema RFID baseado em iot de catraca de onibus publico, para gerencia transação de creditos de cartão mobilidade
feito em cobol com integração ao C++ com interface gráfica em html

- SImulação de credito real
- sistema de aprovação
- Validação de credito a partir da linha do onibus
- Confirmação na tela do usuario de credito usado
- Numero da linha
- Logs de acesso
