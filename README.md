# Leitor_RFID_De_Gerenciamento
Sistema RFID baseado em iot de catraca de onibus publico, para gerencia transação de creditos de cartão mobilidade
feito em cobol com integração ao C++ com interface gráfica em html

- SImulação de credito real
- sistema de aprovação
- Validação de credito a partir da linha do onibus
- Confirmação na tela do usuario de credito usado
- Numero da linha
- Logs de acesso

O projeto [Leitor_RFID_De_Gerenciamento](https://github.com/PyzerFs/Leitor_RFID_De_Gerenciamento.git?utm_source=chatgpt.com) apresenta uma solução baseada em tecnologia RFID para gerenciamento e identificação automatizada de ativos, pessoas ou objetos. Sistemas RFID são amplamente utilizados em automação industrial, controle de estoque, rastreamento logístico e autenticação devido à capacidade de leitura sem contato físico, rapidez operacional e redução de erros humanos. ([S.U.R.F.E.R. RFID][1])

## Justificativa Técnica

O desenvolvimento do projeto é tecnicamente relevante porque propõe uma arquitetura capaz de integrar hardware RFID com software de gerenciamento, permitindo automação de processos operacionais e coleta de dados em tempo real. A utilização de RFID oferece vantagens importantes em comparação a métodos tradicionais de código de barras ou controle manual, como:

* Leitura simultânea de múltiplas tags;
* Identificação sem necessidade de contato direto;
* Maior velocidade de processamento;
* Redução de falhas operacionais;
* Melhor rastreabilidade de ativos e usuários;
* Integração facilitada com bancos de dados e sistemas web.

Bibliotecas e SDKs RFID modernos em Python demonstram que esse tipo de integração possui ampla aplicabilidade em sistemas embarcados, IoT e automação corporativa. ([GitHub][2])

Além disso, projetos RFID possuem forte aderência ao conceito de sistemas inteligentes e Internet das Coisas (IoT), permitindo expansão futura para monitoramento remoto, análise de dados, dashboards em tempo real e controle automatizado de acessos.
